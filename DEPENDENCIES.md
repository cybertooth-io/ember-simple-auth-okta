# Dependencies

```bash
ember install ember-auto-import

ember install ember-cli-addon-docs  # using esdoc

ember install ember-concurrency

# For supporting before Ember-3.10
ember install ember-decorators-polyfill

ember install ember-root-url  # using for resources like img tags: https://github.com/ember-learn/ember-cli-addon-docs/issues/74

ember install ember-simple-auth

yarn add @okta/okta-auth-js

```
