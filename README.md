ember-simple-auth-okta
==============================================================================

Ember Simple Auth implementation wrapping `okta-auth-js`.


Compatibility
------------------------------------------------------------------------------

* Ember.js v2.18 or above
* Ember CLI v2.13 or above


Docs
------------------------------------------------------------------------------

[View the docs here](https://cybertooth-io.github.io/ember-simple-auth-okta/).
